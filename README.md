RGB/Hex Colour Module for Python
================================

A simple class-based method of manipulating colours by themselves, in terms of their red, green and blue components.


Methods of the Colour class
---------------------------
* __str__
* __repr__
* __gt__
* __lt__
* __ge__
* __le__
* __eq__
* __ne__
